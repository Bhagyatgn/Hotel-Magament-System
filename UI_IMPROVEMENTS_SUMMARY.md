## 🚀 To Run the Application

### Frontend

```bash
cd hotel-booking-frontend
npm start
# Runs on http://localhost:3000
```

### Backend

```bash
cd backend
npm install
node server.js
# Runs on http://localhost:5000
```

### Make sure to:

1. Have MySQL running with the `hotel_booking` database
2. Create `.env` file in backend with DB credentials (already done)
3. Frontend `.env` file created with API URL
